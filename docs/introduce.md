# Menu3: Make Finder Work Faster

Menu3 is a Finder menu enhancement tool built for macOS. It streamlines everyday file workflows with quick actions right where you work: copy/move files, copy file names and paths, create folders and text files, save clipboard content into the current directory, open apps in the current location, and jump to specific paths.

## Key Features

- Fast access to common file actions in Finder
- Copy/move workflows with batch-friendly handling
- One-click copy for file names and full paths
- Quick creation of folders and text files
- Save clipboard files or text directly to current folder
- “Open in App” with recent app shortcuts
- Always-available menu bar utility for quick triggering

## Highlights

- **Friendly across all directories**: designed to work consistently across local disks, external storage, iCloud, and cloud-drive folders
- **Smooth cross-storage workflow**: keep the same action logic when moving between local and cloud locations
- **Time-saving by design**: consolidates multi-step Finder operations into one focused menu
- **Built for heavy file users**: ideal for creators, developers, and productivity-focused teams
