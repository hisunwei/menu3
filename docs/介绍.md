# Menu3：让 Finder 操作更高效

Menu3 是一款专为 macOS 打造的 Finder 菜单增强工具，围绕高频文件操作做了更顺手的体验优化。你可以在常用场景中更快完成复制/移动、复制文件名与路径、新建文件夹与文本文件、剪贴板内容保存、在此打开应用、前往指定路径等操作，减少重复点击和窗口切换。

## 主要功能

- Finder 场景下的常用文件操作快捷入口
- 复制/移动文件与批量处理能力
- 一键复制文件名、文件路径
- 快速新建文件夹、新建文本文件
- 粘贴板内容直接保存到当前目录
- “在此打开应用”与最近应用快速调用
- 菜单栏常驻，配合触发方式快速唤起

## 产品亮点

- **全目录友好**：本地磁盘、外接存储、iCloud 与云盘目录尽量保持一致体验
- **跨存储协作顺畅**：在本地与云端目录间切换时，操作逻辑统一，降低学习成本
- **高频动作更省时**：把原本分散在多个步骤的操作收敛到一个菜单中完成
- **适合重度文件用户**：创作者、开发者、办公用户都能显著提升日常效率
